# ⚡ JARVIS X — Ultimate Cyber Matrix Protocol Dashboard

A futuristic **Cyberpunk UI Dashboard** that acts as an all-in-one telemetry center. It integrates high-frequency stock data, live international news wires, and FIFA World Cup tracking with a built-in voice control system—all synced perfectly to **Bangladesh Standard Time (BST)**.

## 🌟 Core Features

*   **🎙️ JARVIS Voice Control:** Built-in Speech Recognition for custom terminal commands (e.g., *"Open YouTube"*, *"FIFA info"*).
*   **🏆 FIFA World Cup 2026 Core:** Active countdown matrix and localized match timings.
*   **📈 Multi-Market Grid:** Real-time tracking frameworks for Dhaka Stock Exchange (DSEX), NASDAQ, and Bitcoin (BTC).
*   **🌍 International News Wire:** Live-updating feed capturing global defense and deep-tech intelligence logs.
*   **📟 Sci-Fi Aesthetics:** Animated Matrix rain canvas, pulsing AI orb, and a distinct neon laser scanning layout.

## 🛠️ Voice Commands to Try
*   `"Open YouTube"` / `"Open Gmail"` — Spawns external secure containers.
*   `"FIFA"` / `"World Cup"` — Triggers audio-visual scheduling telemetry updates.
*   `"Stock"` / `"Market"` — Prompts a broadcast breakdown of market matrices.
